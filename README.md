# Research Principles

This repository follows a structured research approach:

1. Public-source investigation
2. Technical validation where possible
3. Contextual analysis
4. Clear sourcing
5. Documentation-first publishing

Research published here should not be interpreted as attribution, legal findings, or forensic confirmation unless explicitly stated.
